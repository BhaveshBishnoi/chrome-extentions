## How to Use

1. **Navigate to any website**
2. **Click the extension icon** (moon icon) in your toolbar
3. **Click "Toggle Dark Mode"** button
4. **Dark mode is now active!** Click again to disable

## Troubleshooting

**Extension not showing?**
- Make sure Developer Mode is enabled
- Refresh the extensions page

**Dark mode not working?**
- Try refreshing the webpage
- Some complex sites may require multiple toggles

**Can't click the button?**
- Make sure you're on an actual webpage (not chrome:// pages)
